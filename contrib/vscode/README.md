# VS Code / Cursor

- Copia `tasks.json` a `.vscode/tasks.json` del **proyecto zscan** (este repo) o fusiona las entradas en tu workspace.
- Ejecuta **Tasks: Run Task** → `zscan: scan (this folder)` para ejecutar **`scan-all`** y crear un run bajo `zscan-runs/<id>-scan/` en el workspace (HTML, JSON, Markdown por ecosistema).

Para otro repositorio, cambia `--root` o abre esa carpeta como workspace.
